About :

The goal of this project is to use the Wikipedia Api to highlight particularly popular Wikipedia articles and display some interesting data about them.

The website has 4 tabs:

  1. Top Articles: this is the first thing visitors should see
  2. Saved Articles
  3. Activity Chart
  4. About: description of the project and references to libraries and APIs used.

Some of the other Api's and references we used were:

  • Get the most viewed articles for a project
  • Get a text extract and thumb summary of a page
  • Get pageview counts for a page
